## Energy Consumption Forecasting using Linear Regression
A Machine Learning project that predicts energy consumption using the Linear Regression algorithm.
This project uses historical energy usage data to train a model and forecast future energy consumption.

 ## Project Overview
This project demonstrates how Machine Learning can be used for energy forecasting.
The model learns relationships between different input features such as temperature, humidity, appliances usage, etc., and predicts energy consumption.

## The project includes:
Data preprocessing
Model training
Prediction
Performance evaluation
Data visualization

## Features
Load and process CSV dataset
Train/Test data splitting
Linear Regression model training
Energy consumption prediction
Model evaluation using MAE, MSE, and R² Score
Actual vs Predicted visualization graph
